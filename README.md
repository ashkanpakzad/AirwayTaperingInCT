# Airway Tapering In CT
A MATLAB implementation of acquiring an airway tapering measurement based on work by:

* K. Quan et al., “Tapering analysis of airways with bronchiectasis,” Proc. SPIE 10574, 105742G (2018). https://arxiv.org/abs/1909.06604

Original work can be found: https://github.com/quan14/AirwayTaperingInCT

## Dependencies
* skel2graph3d-matlab: https://github.com/phi-max/skel2graph3d-matlab

## Usage
* clone skel2graph3d under dependencies and add to MATLAB path.
* clone this Repository and add to MATLAB path.
* decompress .rar files and run the mainscript.
